# 静态博客的优缺点

静态博客因其简洁和高效而受到许多开发者和技术写作者的青睐。

## 优点

### 1. 速度快

纯静态文件，CDN 直接分发，加载速度极快。

### 2. 安全

没有数据库，没有后端代码，攻击面积极小。

### 3. 部署简单

GitHub Pages、Vercel、Netlify 均可免费托管。

## 缺点

### 1. 更新流程繁琐

每次写文章都要运行构建脚本。

### 2. 动态内容受限

评论、搜索等功能需要引入第三方服务。

## 结论

对于技术博客来说，静态博客是一个非常不错的选择。
