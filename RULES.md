# Claw Blog - 设计规范

## 1. 概念与愿景

一个极简的静态博客系统，灵感来自传统书本的阅读体验。使用温暖的米黄色纸张质感背景，配合深灰色正文字体，营造出纸上阅读的宁静感。所有字体采用衬线体，强调文字本身的韵律与重量。

## 2. 设计语言

### 配色方案
```
背景色:     #FAF6F1   (淡米黄色，温暖纸张质感)
正文色:     #333333   (深灰色，柔和不刺眼)
标题色:     #1A1A1A   (近黑色，强调层次)
链接色:     #6B5B4F   (棕灰色，融入整体)
边框/分割:  #E5DDD3   (淡褐线，页面区域分隔)
悬停强调:   #C4A77D   (暖金色，交互反馈)
```

### 字体
```
正文/标题: "Noto Serif SC", "Source Han Serif CN", Georgia, serif
代码:      "JetBrains Mono", "Fira Code", Consolas, monospace
```

### 间距系统
```
基础单位: 8px
小间距:   8px / 16px
中间距:   24px / 32px
大间距:   48px / 64px
```

## 4. 功能逻辑

### 4.1 BLOGS.py 渲染脚本

**输入：** `blogs/` 目录下的 `.md` 文件

**输出：**
- `blogs/[slug].html` — 完整独立 HTML 页面
- `blogs.json` — 博客目录索引

**Front Matter 格式：**
```yaml
---
title: "博文标题"
date: "2024-01-15"
description: "博文摘要"
---
```

**MENU.json 结构：**
```json
[
  {
    "title": "博文标题",
    "date": "2024-01-15",
    "description": "博文摘要",
    "slug": "hello-world"
  }
]
```

### 4.2 渲染流程

1. 扫描 `blogs/` 目录下所有 `.md` 文件
2. 解析每个文件的 front matter
3. 将 md 内容转为完整 HTML 页面（注入模板）
4. 生成 `MENU.json` 包含所有博文元数据
5. 输出到 `blogs/` 和 `blogs.json`

### 4.3 页面渲染

**index.html:**
- 页面加载时通过 JS fetch `blogs.json`
- 按 date 倒序渲染目录列表
- 点击条目跳转到 `blogs/[slug].html`

**article.html:**
- 解析 URL query param 获取当前文章 slug
- 左侧目录通过 JS 分析 h1/h2/h3 生成（固定锚点导航）
- 返回首页链接指向 `../index.html`

## 5. 组件清单

### 5.1 博客卡片 (index 目录项)
- 默认: 白色背景，淡边框，标题 + 日期 + 摘要
- 悬停: 背景微微加深，标题颜色变为暖金色
- 点击: 跳转对应博文页

### 5.2 导航栏 (article 页顶栏)
- 左侧: "← 返回首页" 链接
- 中间: 当前博文标题
- 固定在顶部

### 5.3 侧边目录
- 显示 h1/h2/h3 标题列表
- 当前滚动位置对应标题高亮
- 点击跳转对应章节

### 5.4 博文正文
- 段落: 行高 1.8，字间距正常
- 标题: h1 32px, h2 24px, h3 20px
- 代码块: 背景 #F5F0E8，带圆角
- 引用: 左侧 3px 暖金色边框
- 链接: 棕灰色，下划线，悬停变色

## 6. 技术栈

- **渲染脚本:** Python 3 (无外部依赖，使用内置库)
- **Markdown 解析:** Python markdown 库 (`pip install markdown`)
- **页面模板:** `template.html` 纯静态 HTML + CSS + JS
- **CDN 依赖:** 无（所有资源内联或本地）
